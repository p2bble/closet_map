개인정보처리방침

본 방침은 옷장지도(ClosetMap) 앱(이하 "앱")이 사용자의 개인정보를 어떻게 수집·이용·보호하는지 안내합니다.

1. 수집하는 정보

앱은 다음 기능을 위해 아래 권한을 사용합니다.

  • 카메라 – 옷 또는 보관 장소 사진 촬영 (사용자가 직접 실행 시에만 접근)
  • 저장소(갤러리) – 기기에 저장된 사진 선택

촬영·선택된 사진은 기기 내부 저장소에만 저장되며, 외부 서버로 전송되지 않습니다.
단, AI 자동 분류 기능 사용 시 선택한 이미지가 Google Firebase AI(Gemini API)로 전송되어 옷 종류를 분석한 후 즉시 삭제됩니다.

2. 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보
  • 계정·로그인 정보

3. 제3자 제공

앱은 다음 제3자 서비스를 사용합니다.

  • Google Firebase AI Logic – AI 분류 기능 (이미지 일시 처리 후 저장하지 않음)
  • Google Firebase App Check – API 무단 접근 방지

위 서비스 외에는 어떠한 제3자에게도 사용자 데이터를 제공하지 않습니다.

4. 데이터 보관

앱에 입력한 모든 데이터(보관 장소, 옷 목록, 보관 기록)는 사용자 기기의 로컬 데이터베이스에만 저장됩니다. 앱 삭제 시 모든 데이터가 함께 삭제됩니다.

5. 아동 개인정보

앱은 만 14세 미만 아동으로부터 의도적으로 개인정보를 수집하지 않습니다.
